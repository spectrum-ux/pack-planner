# 🧳 Pack-Planner.AI

Climate-smart packing planner for any destination. Single HTML file — no build tools, no backend.

## Features

- 100+ cities with real monthly temperature data
- Start/end date pickers with live temperature preview
- 3 bag types: Small carry-on (34L), Medium check-in (45L), Large check-in (70L)
- Selectable bag toggles — use any combination
- Tap-to-move on mobile, drag-and-drop on desktop
- Interactive checklists with progress tracking
- Email export (2 addresses)
- Apple Notes checklist export
- Native share sheet support (iOS/Android)
- PWA-ready — add to Home Screen
- Works fully offline after first load

## Tech Stack

Single `index.html` — React 18 + Tailwind CSS + Babel (all via CDN). Zero dependencies to install.
